# Compliance Statement / 合规声明

**Semantic Shield v1.4.0**

---

## Narrow compliance boundary / 精简合规边界

We **only** enforce these limits in code:

| Prohibited / 禁止 | Allowed / 允许 |
|-------------------|----------------|
| Bypass server-side login, registration, VIP, paid subscription | Client-side copy/cut/paste/contextmenu unblock |
| Force-hide login or payment modals | Highlight hidden fake close buttons |
| | Block nuisance ads, auto popups, corner floaters (non-access-control) |
| | Unsubscribe flow shortcut (email opt-out pages) |
| | Restore scroll + click native close on non-paywall dialogs |
